## SafetySpot_profil.png

This screen shows the personal **"Profil"** (User Profile) and settings control hub.

* **Header:** Features the section title *"Profil"* and a settings gear wheel icon in the upper right corner.
* **User Identity Card:** A large rounded block at the top displays:
* A circular illustrated avatar of the cartoon student character.
* The user's full name, **"Max Mustermann"**.
* The current level marker, **"Level 12"**, backed by a green progress bar reading *"2.450 / 3.000 XP"*.
* Three split status columns beneath the identity text mapping core achievements: **2.450** *Punkte* (Points), **7** *Tage Streak* (Day Streak), and **24** *Abzeichen* (Badges).

* **Navigation List Items:** Divided into clear functional blocks, each row featuring a right navigation arrow (`>`):
* **Progress Block:** Contains *Mein Fortschritt* (My Progress), *Abzeichen* (Badges), and *Meine Szenarien* (My Scenarios).
* **Support Block:** Contains *Einstellungen* (Settings) and *Hilfe & Feedback* (Help & Feedback).
* **Session Control:** A standalone row at the bottom with red text and an exit icon labeled **"Abmelden"** (Log out).

* **Navigation Bar:** The bottom tab bar has the **Profil** tab active (highlighted in green with a person silhouette icon).
