## SafetySpot_home.png

This screen represents the main dashboard or **"Start"** view for an authenticated student.

* **Header:** Displays a personalized greeting, *"Hallo, Max! ??"*, in a bold dark font. To the right sits a notification bell icon featuring a red alert dot.
* **User Status Section:** Two main metric cards are placed side-by-side:
* **Level Card (Left):** Shows a green starburst badge containing the number **"12"**, captioned with the rank *"Sicherheitsprofi"* (Safety Professional).
* **Points Card (Right):** Shows a yellow star icon alongside the user's current score of **"2.450"** points.
* **Streak Tracker:** Positioned underneath these cards, a row displays a fire icon (**"Streak"**) and a checkmark stating *"V 7 Tage in Folge"* (7 days in a row).

* **"Weitermachen" (Continue) Section:** A specialized banner allows the user to resume their last activity. It features a blue beaker graphic, the title *"Chemieraum"*, the sub-topic *"Gef„hrliche Stoffe"*, a horizontal progress bar filled to **60%**, and a green **"Fortsetzen"** (Continue) action button.
* **"Kategorien" (Categories) Section:** Displays a grid layout of topic blocks with an *"Alle anzeigen"* (Show all) shortcut link on the right. The visible categories include:
* **Chemie** (Blue background, beaker icon, 8 Scenarios)
* **Werkraum** (Orange background, crossed tools icon, 6 Scenarios)
* **Sport** (Green background, lifebuoy/sports icon, 7 Scenarios)
* **Technik** (Purple background, gear icon, 5 Scenarios)

* **Navigation Bar:** A standard bottom tab bar with four positions: **Start** (active, highlighted in green with a home icon), *Szenarien*, *Ranking*, and *Profil*.
